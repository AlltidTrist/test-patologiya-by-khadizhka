# Сайт для тренировки с тестами по патологической анатомии

Веб-сайт для тренировки с 522 вопросами из двух курсов: общего и частного.

## Файлы проекта

- `index.html` - главная страница сайта
- `styles.css` - стили оформления
- `script.js` - JavaScript логика
- `questions.json` - файл с вопросами и вариантами ответов
- `parse_simple.py` - скрипт для извлечения вопросов из Word файлов

## Как использовать

1. Откройте файл `index.html` в браузере
2. Выберите категорию (Общий курс, Частный курс или Все категории)
3. Выберите вариант ответа на вопрос
4. Сразу увидите, правильный ли ответ
5. Используйте кнопки "Предыдущий" и "Следующий" для навигации

## Важно: Правильные ответы

В текущей версии правильные ответы не указаны в файле `questions.json`. 

**Чтобы указать правильные ответы:**

1. Откройте файл `questions.json`
2. Для каждого вопроса найдите поле `"correct"` (сейчас оно `null`)
3. Укажите букву правильного ответа (например, `"a"`, `"b"`, `"c"` и т.д.)

Пример:
```json
{
  "question": "Вопрос...",
  "options": [...],
  "correct": "a",  // <-- укажите правильный ответ здесь
  "category": "общий курс"
}
```

**Временное решение:** Если правильный ответ не указан, по умолчанию считается правильным первый вариант (a). Это нужно изменить вручную в JSON файле.

## Технические детали

Сайт использует чистый JavaScript без дополнительных библиотек. Все данные загружаются из файла `questions.json` через Fetch API.

Для локального использования может потребоваться локальный веб-сервер из-за политики CORS браузеров. Можно использовать:
- Python: `python -m http.server 8000`
- Node.js: `npx http-server`
- Или просто открыть в браузере (может работать не во всех браузерах)
